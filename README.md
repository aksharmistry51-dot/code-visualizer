# Code Visualizer + Debug Explainer

An AI-powered web app that analyzes JavaScript code step-by-step.

## Features
- Step-by-step execution visualization
- Variable tracking at each step
- Error detection with simple explanations
- Code fix suggestions
- Execution flow summary

## Live Demo
https://code-visualizer-theta.vercel.app

## Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express
- AI: Groq API (Llama 3)
- Deployed: Vercel + Render

## How to Run Locally

### Backend
```bash
cd backend
npm install
node server.js
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Environment Variables
Create a `.env` file in the backend folder:
GROQ_API_KEY=your_groq_api_key
PORT=5000
